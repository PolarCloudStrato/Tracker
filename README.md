# Tracker
Moiture intrusions observed by satellites


On va essayer de garder une trace propre du code que je vais faire, avec des sauvegardes et des commentaires
Ce projet va se séparer en 4 étapes: 
1. Capturer les intrusions de vapeur d'eau (seuil simple d'IVT --> plus complexe avec axe comme Spensberger 2024)
2. Reconstruire peut-être les trajectoires à partir de ces cas d'intrusions
3. Colocalisations avec les traces EarthCARE
